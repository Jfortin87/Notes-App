# 📝 Notes App — Step 1 (UI & UX Upgrade)

## 📌 Overview

This project is a simple Notes App built with **HTML, CSS, and JavaScript**.
The `step-1` branch focuses on improving the **user interface and user experience**, transforming a basic functional app into a more polished, modern design.

---

## 🚀 Features

### ✨ UI Enhancements

* Full-page gradient background
* Glassmorphism-style container (blur + transparency)
* Larger, centered application layout
* Title positioned at the top for better structure

### 📦 Improved Layout

* Expanded container width for better readability
* Responsive design (works on smaller screens)
* Clean spacing and alignment

### 📝 Better Note Display

* Notes now use a **stacked card layout**
* Supports **longer notes without breaking layout**
* Preserves line breaks and formatting
* Improved readability with spacing and typography

### 🎯 User Experience Improvements

* Larger input area for writing notes
* Resizable textarea
* Smooth hover and button interactions
* Cleaner action buttons (Edit/Delete)

---

## 🛠️ Tech Stack

* **HTML5** — Structure
* **CSS3** — Styling, layout, animations
* **JavaScript (Vanilla)** — App functionality
* **LocalStorage API** — Persistent data storage

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/notes-app.git
   ```

2. Navigate to the project folder:

   ```bash
   cd notes-app
   ```

3. Open `index.html` in your browser.
